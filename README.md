## Android приложение для помощи игрокам и мастерам для игры в D&D 5e.

## Возможности
* список заклинаний
* бестиарий
* закладки для заклинаний и монстров.
* поиск по наимновани заклинания и монстра
* фильтрация по классам, уровням, опастности
